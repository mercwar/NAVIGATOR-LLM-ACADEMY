Chapter 1: Using Interfaces – Buttons and Navigation

Page 3 – Interfaces

Modern LLM platforms provide user interfaces that make interaction simple. Instead of typing commands into a terminal, users can click buttons, navigate menus, and use graphical elements to control the model.

Buttons often trigger specific actions such as submitting a prompt, clearing the chat, or exporting results. Navigation menus help users move between sections like settings, saved conversations, or tutorials.

Understanding these interface elements is essential for beginners. It reduces friction and allows users to focus on learning how the LLM responds rather than struggling with technical details.

Practical Exercise

  • Open your LLM platform and locate the main navigation menu.
  • Identify buttons for submitting prompts and clearing responses.
  • Experiment by clicking each button and observing the effect.
  • Note how navigation helps you move between different sections.

By practicing with buttons and navigation, you’ll gain confidence in using the interface effectively. This skill prepares you for more advanced workflows later in the book.
